Enunciado da Atividade

Você foi contratado para desenvolver um sistema simples de cadastro escolar. 
Esse sistema deverá utilizar herança para representar diferentes tipos de pessoas dentro da escola e construtores para inicializar os objetos corretamente.

Requisitos:

1. Crie uma classe Pessoa que possua os seguintes atributos:

    nome (String)
    idade (int)

   Além disso, a classe deve conter:

    Um construtor que receba nome e idade como parâmetros.
    Um método exibirInformacoes() que mostre os dados da pessoa.



2. Crie uma classe Aluno, que herda da classe Pessoa e possua um atributo adicional:

    matricula (String)

   Essa classe deve conter:

    Um construtor que receba nome, idade e matricula, chamando o construtor da classe Pessoa com super.
    O método exibirInformacoes() deve ser sobrescrito (@Override) para também mostrar a matrícula.



3. Crie uma classe Professor, que também herda da classe Pessoa, com um atributo adicional:

    disciplina (String)

   Essa classe deve conter:

    Um construtor que receba nome, idade e disciplina, chamando o construtor da classe Pessoa com super.
    O método exibirInformacoes() deve ser sobrescrito para mostrar também a disciplina.



4. Crie uma classe principal chamada Escola que contenha o método main().

    No método main, crie pelo menos um objeto de cada classe (Pessoa, Aluno, Professor).
    Chame o método exibirInformacoes() de cada objeto para mostrar os dados no console.
